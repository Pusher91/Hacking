Port Forwarding and Tunnelling

SSH Tunnelling

  • Local port forwarding
    • Send traffic from port on local machine to port on remote machine
    • ssh <username>@<ssh server> –L <local port>:<remote host>:<remote port>
  • Remote port forwarding
    • Send traffic sent to port on remote machine to our local machine
    • ssh <username>@<ssh server> -R <local port>:<remote host>:<remote port>
  • Dynamic port forwarding
    • Forward open ports on victim to attacker
    • ssh <username>@<ssh server> -D <address to bind to>:<port to bind to>
